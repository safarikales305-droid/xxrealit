# Shorts hudba (rozšíření)

Aktuálně generátor používá **bezpečné lavfi** signály (`demo_soft`, `demo_warm`, `demo_pulse`) — žádné binární MP3 v repu.

Sem můžete později přidat vlastní **royalty-free** skladby (např. `track1.mp3`) a v `ListingShortsFromPhotosService` je namapovat podle `musicKey`.
